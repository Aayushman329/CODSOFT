# Tasks Completed

## Task 1 – Number Game

### Description
A simple number guessing game where the program generates a random number and the user tries to guess it.

### Features
- Random number generation between 1–100
- User input for guessing the number
- Feedback if guess is **too high or too low**
- Limited number of attempts
- Option to play multiple rounds
- Score tracking

### Concepts Used
- Java Random class
- Loops
- Conditional statements
- User input using Scanner

---

## Task 2 – Student Grade Calculator

### Description
This program calculates the **total marks, average percentage, and grade** of a student based on marks entered for multiple subjects.

### Features
- Input marks for multiple subjects
- Calculate total marks
- Calculate average percentage
- Automatic grade calculation
- Display results clearly

### Concepts Used
- Loops
- Conditional statements
- Basic arithmetic operations
- User input handling

---

## Task 3 – ATM Interface

### Description
A console-based ATM simulation where users can check balance, deposit money, and withdraw money.

### Features
- Check account balance
- Deposit money
- Withdraw money with balance validation
- Menu-driven interface
- Separate classes for **ATM** and **Bank Account**

### Concepts Used
- Object-Oriented Programming (OOP)
- Classes and Objects
- Methods
- Encapsulation
- Conditional logic
